🔥 Firebase Setup

Connect this system to your Firebase project. This is a one-time setup — paste your Firebase config below.

1 Create a Firebase project

Go to console.firebase.google.com → New Project → disable Google Analytics → Create.

2 Enable Firestore

In your project → Build → Firestore Database → Create database → Start in test mode → Choose a region → Done.

3 Enable Email/Password Auth

Build → Authentication → Get Started → Email/Password → Enable → Save.

4 Get your config

Project Settings (gear icon) → General → Your apps → Web app (</>) → Register → Copy the firebaseConfig values below:

💡 Your config is saved in your browser's localStorage and never sent anywhere. Only Firebase itself is contacted. Firebase free tier allows 1GB storage and 50K reads/day — more than enough for this system.
